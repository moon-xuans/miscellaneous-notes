Linux
1.目录切换
	cd usr:切换到该目录下usr目录
	cd ..:切换到上一层目录
	cd /:切换到系统根目录
	cd ~:切换到用户主目录
	cd -:切换到上一个操作所在目录
2.目录的操作命令（增删改查）
	mkdir 目录名称:增加名称
	ls -l(ll):查看目录信息
	find 目录：寻找目录
	mv 目录名称 新目录名称：修改目录的名称
	mv 目录名称 目录的新位置：移动目录的位置
	cp -r 目录名称 目录拷贝的目标位置：拷贝目录（-r 表示递归）
	rm 目录：删除目录
3.文件的操作命令（增删改查）
	touch 文件名称：创建名称
	cat/more/less/tail 文件名称:文件的查看
		命令名称：touch
		功能：新建文件

		命令名称：cat
		功能：显示文件内容
		参数：
		-n 显示行号
		其他：如果文件信息较长，则只会显示后几行。

		文件处理命令
		命令名称：more
		功能：分页显示文件内容
		其他：
		空格或f 翻页
		回车 下一行
		q或Q 退出

		命令名称：less
		功能：分页显示文件内容（可向上翻页）
		其他：
		page up 向上翻页
		上箭头 上一行
		/关键词 搜索
		n 向下继续搜索

		命令名称：head
		功能：显示文件前几行
		用法：head -n 多少行 文件名

		命令名称：tail
		功能：显示文件后几行
		用法：tail -n 多少行 文件名
			
		
	vim 文件：修改文件的内容
	rm -rf:删除文件
4.压缩文件的操作命令
	tar -zcvf 打包压缩后的文件名 要打包压缩的文件：打包并压缩文件
	tar -xvf 压缩文件：解压压缩包
	
	v：  详细信息
	f：  filename 要操作的文档名
	
	z     用于gzip压缩
	c:   创建文档
	
	x：将压缩好的文件解压打开
5.Linux的权限命令
	文件的类型：
		d:代表目录
		-:代表文件
		|：代表软连接
	权限分为以下几种：
		r:代表可读 	4
		w:代表可写  2 
		x:代表可执行 1
	用户类型：
		u：所有者
		g：文件所在组
		o: 其他组
	chmod u=rwx,g=rw,o=r aaa.txt 或者 chomod 764 aaa.txt：修改文件/目录的权限的命令
	
6.用户管理
	useradd 选项 用户名：添加用户账号
	userdel	选项 用户名：	删除用户账号
	usermod 选项 用户名：修改账号
	passwd 用户名：更改或创建用户的密码
	passwd -S 用户名：显示用户账号密码信息
	passwd -d 用户名：清除用户密码
7.用户组的管理
	groupadd 选项 用户组：增加一个新的用户组
	groupdel 用户组：要删除已有的用户组
	groupmod 选项 用户组：修改用户组的属性
8.其他常用命令
	pwd：显示当前所在位置
	ps -ef/ps -aux:查看当前系统正在运行进程
	kill -9 进程的pid：杀死进程（-9表示强制终止）
	
	网络通信命令：
		ifconfig:查看当前系统的网卡信息
		ping:查看与某台机器的连接情况
		netstat -an:查看当前系统的端口使用
	
	shutdown:指定现在立即关机
	reboot：重开机
